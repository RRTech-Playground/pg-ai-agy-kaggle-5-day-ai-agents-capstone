# The CEO Co-Pilot Project DNA

## Stack
- **Framework**: Google ADK (Agent Development Kit)
- **Language**: Python
- **Models**: Gemini 3 models

## Core Constraint
- **Strict Zero-Data-Ingestion**: Absolutely no ingestion of live corporate internal systems to preserve absolute privacy. All analysis must be based on explicitly provided input diaries or decision logs.

## Routing
- **Skills Progressive Disclosure**: Index the skills library and force progressive disclosure. Deep references and sub-skills must only be loaded on-demand when explicitly triggered.
